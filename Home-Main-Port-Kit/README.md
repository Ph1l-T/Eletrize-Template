# Home Main Port Kit

Kit técnico para portar a página inicial premium (Main) para outro app.

Conteúdo:
- `docs/`: prompt técnico, contratos de dados e checklist de integração.
- `src/`: template HTML + CSS + JS base para implementação.
- `assets/`: ícones e imagem obrigatórios para deixar a tela visualmente igual.

Objetivo:
- Replicar exatamente a Home com:
  - widget de clima + saudação dinâmica,
  - card now playing com controles,
  - card último ambiente,
  - card dispositivos ligados agrupado por ambiente.
